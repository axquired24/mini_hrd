## Sistem HRD Mini
---
Sistem simpel untuk pembelajaran yang terdiri dari fitur :

 - Melihat Daftar Karyawan Per Divisi
 - Melihat Daftar Semua Divisi
 - Pencarian Data & Gaji Karyawan (hitungan perhari) dari awal masuk hingga hari ini
